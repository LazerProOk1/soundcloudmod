## Зачем это нужно?

**Обычный SoundCloud в браузере** — тормоза, реклама, нет горячих клавиш, нет офлайна, жрёт RAM.

**Яндекс Музыка** — огромная библиотека, но нет инди и андеграунда, региональные ограничения, нужна подписка.

**SoundCloud Desktop** — нативное приложение с полным доступом к 60+ млн треков, офлайн-режимом и всем удобством десктопного приложения. Бесплатно.

## Возможности

### Плеер
- Полноценный плеер: очередь, история, перемотка, громкость
- **Мини-плеер** — компактное окно поверх всех остальных, перетаскивается
- Кроссфейд между треками
- Эквалайзер (10 полос) с пресетами и нормализацией громкости
- Медиа-клавиши клавиатуры (пауза, следующий, предыдущий)
- Управление из системного трея

### Контент
- Лайки, плейлисты, альбомы, страницы артистов
- **SoundWave** — умная лента персональных рекомендаций
- Поиск по трекам, плейлистам, пользователям
- Лирика с визуализатором и синхронизацией
- Плавающие комментарии на шкале прогресса

### Офлайн и импорт
- Кэширование аудио для мгновенного переключения
- **Офлайн-режим** — скачивай любимые треки и слушай без интернета
- **Импорт из Яндекс Музыки** — перенеси свою библиотеку прямо в SoundCloud Desktop

### Интеграции
- Discord Rich Presence — показывает что слушаешь
- **Direct Mode** — работа напрямую через SoundCloud API без посредников

### Интерфейс
- Тёмная тема с акцентным цветом (5 пресетов + свой)
- Эффект жидкого стекла (Liquid Glass UI)
- Кастомный фон: своё изображение + блюр + прозрачность
- Русский и английский интерфейс

## Сборка из исходников

```bash
# Установить зависимости
pnpm install

# Режим разработки
npx tauri dev

# Production-сборка
npx tauri build
```

Требуется: Node.js 20+, pnpm, Rust toolchain.
