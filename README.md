# SoundCloud Desktop

Неофициальный десктопный клиент SoundCloud для Windows, macOS и Linux на базе Tauri 2 + React.

## Возможности

- Полноценный плеер: очередь, история, перемотка, громкость
- Мини-плеер — компактное окно поверх остальных, перетаскивается
- Эквалайзер (10 полос) с пресетами и нормализацией громкости
- Кроссфейд между треками
- Discord Rich Presence
- Кэширование аудио для быстрого переключения
- Плейлисты, альбомы, страницы артистов
- Soundwave — умная лента рекомендаций
- Лирика с визуализатором
- Тёмная тема с акцентным цветом на выбор (5 пресетов + свой)
- Кастомный фон (изображение + блюр + прозрачность)
- Поддержка русского и английского интерфейса
- Плавающие комментарии на шкале прогресса

## Стек

| Слой | Технология |
|------|-----------|
| Обёртка | Tauri 2 (Rust) |
| UI | React 19 + TypeScript |
| Стили | Tailwind CSS v4 |
| Состояние | Zustand |
| Бэкенд | Rust (Axum) |

## Сборка

```bash
# Зависимости
pnpm install

# Dev-режим
pnpm tauri dev

# Production-сборка
pnpm tauri build
```

Требуется: Node.js 20+, pnpm, Rust toolchain.
